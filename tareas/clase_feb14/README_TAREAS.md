---

### Descripción breve

Debes resolver el problema **Two Sum** en LeetCode:

- Recibirás un arreglo de enteros `nums` y un entero `target`.
- Debes devolver los **índices** de dos números que sumen exactamente `target`.
- Puedes asumir que existe una única solución y que no debes usar el mismo elemento dos veces.

Es un ejercicio ideal para practicar fuerza bruta vs solución con hash map y para repasar el análisis de complejidad (O(n²) vs O(n)).
